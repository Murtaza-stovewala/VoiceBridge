# **App Name**: VoiceBridge

## Core Features:

- Voice Input and Transcription: Capture user's spoken queries, transcribing them to text for processing. Also implements a text fallback for users uncomfortable speaking.
- AI-Powered Blocker Identification: Use AI to analyze the transcribed query and available data to identify potential blockers preventing the user from accessing government schemes. Uses a tool to identify whether the LLM should reason about the impact of different facts, and incorporate the most critical and relevant ones in its output.
- Step-by-Step Guidance Generation: Generate actionable, step-by-step instructions to help the user overcome identified blockers, providing clear and simple guidance. These instructions are shown on the user interface in simple terms
- Multilingual Support: Offer support for multiple languages to cater to diverse user base. Uses language auto-detection to pre-set the desired language
- Privacy-Focused Interaction: Ensure user data privacy with consent-based interactions and session-based data handling.
- Help and Community Support: Integrate help resources and community support options for users requiring additional assistance.

## Style Guidelines:

- Primary color: A muted cyan (#77B5FE) to convey trust and calm.
- Background color: Light cyan (#E5F2FF) for a clean and accessible feel.
- Accent color: Soft blue (#8FB4F7) to highlight key actions.
- Body and headline font: 'PT Sans', a humanist sans-serif, for readability and a touch of warmth.
- Mobile-first, large buttons, high contrast, minimal scrolling for ease of use.
- Use clear and universally understandable icons to represent key functions.
- Subtle animations and transitions to guide users through the conversation flow without being distracting.
